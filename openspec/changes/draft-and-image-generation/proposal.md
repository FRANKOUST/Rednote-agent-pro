## Why

The product only becomes useful when it can transform insights into concrete draft assets and matching image outputs.

## What Changes

- Add draft generation provider interfaces and default implementation.
- Add image generation provider interfaces and default implementation.
- Persist drafts and image assets.

## Capabilities

### New Capabilities

- `draft-generation`: AI or mock generation of structured content drafts.
- `image-generation`: AI or mock generation of local image assets.

### Modified Capabilities

None.

## Impact

- Creates the assets that feed review and publish flows.
