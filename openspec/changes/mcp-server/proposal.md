## Why

The product vision includes AI-tool access, so the platform needs an MCP-style tool surface over shared use cases.

## What Changes

- Add MCP tool discovery and invocation endpoints.
- Expose core use cases as reusable tools.

## Capabilities

### New Capabilities

- `mcp-tool-surface`: Tool discovery and tool invocation over shared services.

### Modified Capabilities

None.

## Impact

- Extends the platform from a web app into an AI-agent-compatible system.
