## Why

The platform needs a programmatic operator interface for integration, automation, and testing.

## What Changes

- Add REST endpoints for runs, drafts, publish jobs, sync records, and audit logs.
- Add shared request validation and response conventions.

## Capabilities

### New Capabilities

- `operator-rest-api`: Shared service-backed HTTP endpoints for the platform.

### Modified Capabilities

None.

## Impact

- Enables external automation and integration testing.
