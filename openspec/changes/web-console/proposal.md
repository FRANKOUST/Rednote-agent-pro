## Why

Operators need a simple console to run the pipeline, review drafts, and inspect activity without using raw APIs.

## What Changes

- Add server-rendered dashboard UI.
- Add forms for pipeline launch, approval, and publish actions.
- Surface recent audit activity.

## Capabilities

### New Capabilities

- `operator-web-console`: Minimal internal dashboard for operating the platform.

### Modified Capabilities

None.

## Impact

- Makes the system demoable and usable by non-developer operators.
